## simple example for node-fetch

### how to execute
	# install package
	$ npm install
	
	# execute
	$ npm run start
	# execute another way
	$ node index.js


### how to construct this project

	# create directory
	$ mkdir node-fetch-example
	
	# initial package.json
	$ npm init
	
	# install node-fetch
	$ npm install node-fetch --save
	
	# add `npm start` script
	
	# add index.js
